# User Management

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/NuneMazmanyan/user-management.git
cd user-management
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).


## What you can do 
- Fetch users on page load with loading and error states (test via DevTools → Network → Throttling: use **Slow 4G** to see loading, **Offline** to see the error state)
- Search by name or email
- Filter by city
- Sort by name (ascending / descending)
- View full user details in a modal
- Edit user names locally by clicking on the name or edit button
